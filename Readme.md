## Initialization
```bash
make init
```
## Dosbox
To setup dosbox you need to enter these commands:
```bash
mount c (directory to the code, etc. /home/pijus/Assembly/tasks/lab1)
c
```
## Compile Assembly code
```bash
yasm programName.asm -o programName.com
programName.com
```
